# 姿勢改善プランナー【無料】自動作成ツール - 技術仕様書

## 概要

**サービス名**: Posture Improvement Planner
**バージョン**: 1.2.0
**更新日**: 2026-06-06
**URL**: https://appadaycreator.com/posture-improvement-planner/

デスクワーク時間・痛みの部位から最適な姿勢改善エクササイズプランを作成。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 1日のデスクワーク時間をセレクトボックスで選択する
2. 気になる痛みや不調の部位にチェックを入れる（複数選択可）
3. 姿勢の悩みと運動習慣を選択する
4. 「姿勢改善プランを作成する」ボタンをクリックして診断を開始する
5. 表示された姿勢スコアとエクササイズプランを毎日のルーティンに取り入れる

## 計算ロジック（v1.1.0）

### 姿勢リスクスコア算出
| 要素 | 配点 |
|------|------|
| デスクワーク1〜2h | +15点 |
| デスクワーク3〜4h | +30点 |
| デスクワーク5〜6h | +50点 |
| デスクワーク7h以上 | +70点 |
| 痛みの部位（目以外）| +10点/部位 |
| 痛みの部位（目の疲れ）| +5点 |
| 姿勢タイプ（複合・スマホ首）| +20/+15点 |
| 運動習慣（週1-2/週3+）| -5/-15点 |

### リスク判定
- 0〜30点: 低リスク（緑）
- 31〜60点: 中リスク（橙）
- 61〜100点: 高リスク（赤）

### エクササイズデータベース
- 首・肩: 3種目
- 腰: 3種目
- 目の疲れ・頭痛: 3種目
- 膝・股関節: 3種目
- 全身疲れ: 2種目
- 猫背: 3種目
- 反り腰: 3種目
- スマホ首: 3種目
- 骨盤ゆがみ: 3種目
- 複合: 2種目

最大8種目をパーソナライズして表示。

## よくある質問（FAQ）

**Q: 姿勢改善プランナーは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 計算結果は正確ですか？**

参考値としてご活用ください。重要な判断の際は専門家・公式機関にご確認ください。

**Q: 何回でも計算できますか？**

はい、回数制限なく何度でもご利用いただけます。条件を変えて比較にお使いください。

**Q: 入力したデータはどこかに保存されますか？**

いいえ。すべての計算はブラウザ内で処理され、データのサーバー送信は行いません。

**Q: スマートフォンで利用できますか？**

はい、PC・スマートフォン・タブレットすべてに対応しています。
